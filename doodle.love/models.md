# Models

Currently, the following objects are available:

- Wall
- Player
- Person
- Item
- Goal

Wall
----

```
wall:x y
```

This will represent a wall.

Player
------

```
player:x y
```

Person
------

```
person:x y update_rate
```

Item
----

```
item:x y
```

Door
----

```
door: x y destiny
```

Goal
----

```
goal: x y required_items
```
